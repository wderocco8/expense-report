# Tasks

Lightweight agile board for tracking work. Managed collaboratively with any local coding agent.

## Files

| File                       | Purpose                                               |
| -------------------------- | ----------------------------------------------------- |
| [`active.md`](active.md)   | Currently in-progress work (keep this short)          |
| [`backlog.md`](backlog.md) | Prioritized upcoming work, organized by priority tier |
| [`archive.md`](archive.md) | Completed tasks, most recent first                    |

## Workflow

1. New tasks → `backlog.md` under the appropriate priority section
2. Starting work → move item to `active.md`
3. Done → move item to `archive.md` with a completion date header

## Notes

- Using this folder for AI-managed task tracking instead of Notion
- Critical bugs should be prioritized over new features
- BACKLOGGED items are acknowledged but intentionally deferred
